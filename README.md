**Auto File Transfer**
**Overview**

~Auto File Transfer is a lightweight Windows desktop utility that automatically monitors a source folder and transfers files to a destination folder in real time. Once configured, the application runs silently in the background and requires no manual intervention.
~The tool is designed for reliable, long-running operation and is suitable for image transfer, data backup, log collection, and automated file workflows.

**Key Features**

~Real-time folder monitoring
~Automatic file transfer
~Background system tray operation
~Resume transfer after system restart
~Safe copy with file size verification
~Duplicate file detection and skip
~One-click reset and reconfiguration
~Lightweight and stable performance
~Internet-based auto-update support

**How It Works**

~Select a Source Folder
~Select a Destination Folder
~Click Move to start automatic transfer
~The app runs in the background and transfers files automatically
~Use Reset to stop transfer and clear settings
~Reset Behavior
~Stops all ongoing file transfers
~Disables folder monitoring
~Clears saved source and destination paths
~Keeps the application running and ready for reconfiguration
~Auto Resume
~If folders are configured and Reset is not used, the app automatically resumes operation after restart
~Designed for unattended and continuous use

**System Requirements**

~Windows 10 or later
~.NET Framework (as bundled with installer)
~Internet connection (only for updates)

**Updates**

~The application supports automatic internet-based updates. When a new version is published, the app checks for updates and installs them securely without manual reinstallation.

**Usage Notes**

~Ensure source and destination folders exist and are accessible
~Do not delete destination files manually during active transfer
~Use Reset before changing folders
